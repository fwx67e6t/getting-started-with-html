Part 2 - HTML
-HTML consists of

  1. text content
  2. markup
  3. references to other docs
  4. links to other pages.
-HTML Markup consists of Tags in the following syntax: <tag>content</tag>
  • bold <b>
  • emphasis <em>
-Tag Attributes
  • <tag attr=”valve”>contents</tag>
  • anchor <a> (link) syntax = <a href=”www.reddit.com”>text</a>
  • images <img> syntax = <img src=”url” alt=”text”> (Note this is an example of a “void” tag (no closing tag.)
-Whitespace
  • break <br> (void tag) (Note this is an example of an inline tag.)
  • paragraph <p>content</p> (Note this is an example of a block tag.)
-Container elements contain stuff inside them.
